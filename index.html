HTML의 역사

HTML(HyperText Markup Language)은 월드 와이드 웹(WWW, World Wide Web)의 핵심적인 구성 요소로, 텍스트 기반의 마크업 언어입니다.
물리학자인 팀 버너스 리가 처음 개발했으며, 웹 페이지를 작성하는 데 사용됩니다.

1980년, 팀 버너스 리는 그가 몸담고 있던 CERN의 연구원들끼리 문서를 이용하고 공유하기 위한 체계인 HTML의 원형, "인콰이어"를 제안했습니다.
1989년 그는 인터넷 기반 하이퍼 텍스트 체계를 제안하는 메모를 작성했고, 1990년 말 HTML을 명시하고, 브라우저와 서버 소프트웨어를 작성했습니다.
하지만 이 프로젝트는 CERN의 정식 채택을 받지 못해 자금 지원을 받지 못했습니다.

최초의 규격

HTML 최초의 공개 설명은 1991년 버너스 리가 처음으로 인터넷에서 문서를 "HTML 태그"로
부르면서 시작되었습니다.
그는 머릿글자로 이루어진 20개의 요소를 기술했고, 그 중 13개는 HTML 4 버전에서도 여전히 존재합니다.

연도별 HTML의 진화

1993년 중반~말 : HTML, HTML+가 표준화 제안을 했지만 만료됨.
1995년 말 : IETF가 HTML 작업 그룹을 설립해 HTML 2.0을 완성함.
1997년 초 : HTML 3.2가 W3C 권고안으로 발표됨. 이는 1996년 말에 폐쇄된 IETF와 같이
W3c에 의하여 개발되고 표준화된 첫 버전임.
1997년 말 : HTML 4.0이 W3C 권고안으로 출시됨.
HTML 4.0은 사용이 중지된 요소를 금지하는 Strict, 허용하는 Transitonal,
프레임 관련 요소에서만 허용하는 Frameset으로 세 가지 문서 형태를 제공함.
1998년 초 : HTML 4.0이 버전이 증가되지 않고 조금 수정되어 재발표됨.
1999년 말 : HTML 4.01이 W3C 권고안으로 출시됨.
2014년 말 : HTML 5의 최종 권고안이 확정되어 표준으로 지정됨.
2016년 말 : HTML 5의 버전업인 HTML 5.1이 표준으로 지정됨.
2017년 말 : HTML 5.1의 버전업인 HTML 5.2가 표준으로 지정됨.
2021년 말 : W3C가 공식적으로 HTML 5.1, 5.2, 5.3 표준, 표준화 작업을 모두 폐기하고 WHATWG HTML Living Standard만이 HTML 표준이 됨.
